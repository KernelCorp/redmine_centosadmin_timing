Redmine timing
===========================

##Installation
Clone repository to your redmine/plugins directory
```
git clone git://github.com/olemskoi/redmine_centosadmin_timing.git
```

Install dependencies
```
bundle install
```

Run migration
```
rake redmine:plugins:migrate
```

Restart redmine
